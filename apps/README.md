# `apps/` (placeholder)

This directory will hold the deployable applications of the Dhee_AI monorepo. It is
currently a placeholder so the repository structure matches
[`MONOREPO_STRUCTURE.md`](../MONOREPO_STRUCTURE.md).

Per the Living Specification Rule in [`ENGINEERING_CONSTITUTION.md`](../ENGINEERING_CONSTITUTION.md),
no application may be created here until its design is specified. See
[`MONOREPO_STRUCTURE.md`](../MONOREPO_STRUCTURE.md) and [`ARCHITECTURE.md`](../ARCHITECTURE.md).

Planned examples (not yet implemented): `apps/web` (React + Monaco IDE), `apps/api`
(Node backend), `apps/desktop`, `apps/cli`, `apps/docs`.
