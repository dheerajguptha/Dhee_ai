# `packages/` (placeholder)

This directory will hold the shared libraries and SDKs of the Dhee_AI monorepo. It is
currently a placeholder so the repository structure matches
[`MONOREPO_STRUCTURE.md`](../MONOREPO_STRUCTURE.md).

Per the Living Specification Rule in [`ENGINEERING_CONSTITUTION.md`](../ENGINEERING_CONSTITUTION.md),
no package may be created here until its design is specified. See
[`MONOREPO_STRUCTURE.md`](../MONOREPO_STRUCTURE.md) for the planned package list, naming
conventions ([`NAMING.md`](../NAMING.md)), and dependency/boundary rules.

Planned examples (not yet implemented): `@dhee/core`, `@dhee/ai-provider`, `@dhee/agent-runtime`,
`@dhee/memory`, `@dhee/rag`, `@dhee/plugin-sdk`, `@dhee/mcp`, `@dhee/ui`, `@dhee/config`,
`@dhee/testing`.
